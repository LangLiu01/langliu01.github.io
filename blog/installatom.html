Ubuntu环境下怎么安装Atom编辑器

电脑重装了系统，以前用的工具没有保存下来，，又费了很大的力气去重新安装这些软件，想想也是挺麻烦的， 于是，借鉴网上的做法，我把它作为笔记写在我的博客里。 安装atom的过程是非常简单的，并没有我想象中的那么复杂，你只需要前往atom的官网 http://atom.io 找到download ,有两个版本 .deb 和.npm的，.deb安装较为简单 sudo dpkg -i atom 你只需要一行命令就可以顺利安装atom,而且如果之前有安装过atom,在安装过程中 新版本会自动覆盖旧版本， 非常简单。 .deb 是Debian系统的格式，Ubuntu也可以兼容，.npm适用于RedHat的系统，也可以强行安装在Ubuntu上，但是听说会有兼容问题,参考如下：

For example, RedHat-based systems use .rpm files for representing software packages, while Debian-based systems use .deb files. These file types differ from one another in their design, and cannot be installed outside of their respective type of operating systems (e.g. installing a .deb file on CentOS, or installing a .rpm file on Ubuntu). However, there are ways to port these packages to other types of systems.
自此，安装成功，你只需要一行命令就可以从终端打开atom atom 为了方便以后使用，你可以右键将它添加到your Favorite. 顺带一提，这种通过dpkg 和apt包管理器安装的软件可以在 usr/bin 找到 如果你不确定它的位置可以用 whereis xx命令找到它的具体位置，这在删除不想要的文件的时候尤其有用。
